# 🏏 Cricket Player Performance Analysis

## 📌 Project Overview
This project analyzes cricket player and team performance data to identify key factors that contribute to match wins. It covers batting, bowling, and team-level insights using Python-based data analysis and visualization techniques.

---

## 🎯 Objectives
- Analyze batting performance (runs, strike rate, average)
- Evaluate bowling performance (wickets, economy)
- Compare team performance
- Identify match-winning factors
- Generate visual insights using graphs

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset Features
- Player Name
- Team
- Matches & Innings
- Runs & Strike Rate
- Wickets & Economy
- Role (Batsman/Bowler/All-rounder)
- Match Result (Win/Loss)

---

## 🔄 Process
1. Data Cleaning using Pandas  
2. Data Analysis & Aggregation  
3. Visualization using Matplotlib & Seaborn  
4. Insight Generation  

---

## 📊 Key Insights
- Teams score significantly more runs in winning matches  
- Higher strike rate strongly correlates with wins  
- Players with consistent averages perform better  
- All-rounders contribute heavily to match outcomes  
- Balanced teams win more matches
